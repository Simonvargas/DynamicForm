# Liferaft Fronend takehome

## Getting started
1. Clone this repository 

   ```bash
   git clone https://github.com/Simonvargas/DynamicForm.git
   ```

2. Install dependencies

      ```npm
      npm install || npm i
      ```
3. Open up two terminals

    ```
    npm start one terminal
    python server.py the other
    ```
4. Begin the fun!
